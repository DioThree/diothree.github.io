Tentang Proyek

Habit Tracker dibuat untuk membantu pengguna membangun kebiasaan positif melalui pelacakan progres yang konsisten. Dibangun menggunakan HTML, Tailwind CSS, dan JavaScript dengan fokus pada UI ringan dan responsif.

Fitur Utama

  • Tambah, edit, dan hapus kebiasaan harian
  • Grafik progres otomatis
  • Mode gelap dan terang
  • Penyimpanan data lokal tanpa server
